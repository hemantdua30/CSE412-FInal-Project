# Quiet Study Spot Finder

CSE 412: Final Project
This is an app that helps students find quiet study places around the campus. It helps them save time as it is based on real user feedback.

## Setup

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Run dev server**
   ```bash
   npm run dev
   ```

3. **Open** [http://localhost:3000](http://localhost:3000)

4. **Login** with test account:

   - `adubey27@asu.edu` / `password2`

## Tech Stack

- Next.js 16, TypeScript, PostgreSQL (Supabase), Tailwind CSS

npm run dev    # Development server
